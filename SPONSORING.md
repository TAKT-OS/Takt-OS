# Sponsoring TAKT OS

TAKT OS is open source ([Apache License 2.0](LICENSE)) and maintained by **Masyukov Pavel** ([p.masyukov@gmail.com](mailto:p.masyukov@gmail.com)).

Sponsorship helps fund development, hardware for testing, documentation, and long-term maintenance of the ESP32 industrial stack.

## How to sponsor

| Channel | Link |

| **GitHub Sponsors** | [github.com/sponsors/Developer-RU](https://github.com/sponsors/Developer-RU) |
| **Email** | [p.masyukov@gmail.com](mailto:p.masyukov@gmail.com?subject=TAKT%20OS%20sponsorship) |

On the repository page, use the **Sponsor** button (heart icon) — it reads links from [`.github/FUNDING.yml`](.github/FUNDING.yml).

## What sponsorship supports

- ESP32 boards and lab hardware for CI and field testing  
- New drivers, middleware modules, and recovery/OTA improvements  
- Documentation, examples, and certification-oriented test coverage  
- Bug fixes and review of community pull requests  

## Other ways to help

- [Star the repository](https://github.com/Developer-RU/Takt-OS)  
- Report issues and contribute code ([CONTRIBUTING.md](CONTRIBUTING.md))  
- Attribute TAKT OS in your products ([docs/licensing.md](docs/licensing.md))  

Thank you for supporting industrial open-source firmware on ESP32.
